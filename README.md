# Score Game

An interactive browser-based score game built with vanilla JavaScript.

## How to Play

1. Enter a number in the input field
2. Select a math operation — Add, Subtract, Multiply or Divide
3. Click Submit — image buttons appear
4. Click each image to apply the operation to your score
5. Watch your score update with each click

## Features

- Four math operations — Add, Subtract, Multiply, Divide
- Images hide after clicking to prevent double counting
- Handles edge case — score defaults to 1 for Multiply and Divide when score is 0

## Tech Stack

- HTML5
- Vanilla JavaScript — DOM manipulation, events, functions

## Concepts Practised

- DOM manipulation — getElementById, innerHTML, textContent
- Event handling — onclick
- Loops — dynamic element creation
- Functions — declarations and calls
- Conditional logic — if/else
- Input validation
- parseInt for type conversion

## Run Locally

Open `index.html` directly in any browser. No installation needed.
